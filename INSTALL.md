# Installation

> Installing Framework Factory is very simple. First thing we need to do is ensure that we
> have [Composer](https://getcomposer.org) installed and configured. After that, we will
> want to run the following command from the _root directory_ of our project.

```bash
$ composer require framework-factory/core
```
